# Roadmap

## Спринт 1 — Repo + Docs (done)
Создание двух репо, документация, DNS, Wiki с Mermaid.

## Спринт 2 — Dashboard (wip)
Статический dashboard на GitHub Pages. Моки на 15-20 сотрудников. Иллюстрации. Timeline detail. Activity feed (polling).

## Спринт 3 — Bot
Telegram-бот для сотрудника. Welcome-видео. Расписание задач Day 0 → Day 90. Кнопки ✅/❓/⏰. `/api/events` для дашборда.

## Спринт 4 — Deploy
Dashboard → GitHub Pages (auto). Bot → Contabo VPS 30. Nginx + certbot. TG webhook. Dashboard переключается на real API.

## Спринт 5 — Pre-meeting polish
QR-коды, backup-видео, сценарий демо на 7 минут, репетиция.

---

## Post-demo roadmap

### Спринт 6 — Роли других сотрудников
Подключение в бот бухгалтера, офис-менеджера, специалиста ОТ, сисадмина. Задачи распределяются между ними. Inline-кнопка «взял в работу».

### Спринт 7 — SMTP рассылки
Автоматическая отправка 5 email-шаблонов из процессов (ИТ+АХО, бухгалтер, юристы и т.д.) при создании employee.

### Спринт 8 — 1С:ЗУП коннектор
Выгрузка данных Т2 / приказов из 1С. Заполнение карточки сотрудника автоматически.

### Спринт 9 — Электронная подпись
Интеграция с Диадок или СБИС. Документы на подпись прилетают новичку в бот, подписываются там же.

### Спринт 10 — Multi-tenant SaaS
Отделение клиентских данных, signup-flow, billing. Старт продажи другим компаниям.

### Спринт 11+ — RAG по нормативам
Отдельный продукт: QA по ГОСТам / документам для инженеров и проектировщиков.
